The Build section is for developers extending the Superform protocol. You can write permissionless hooks to compose new multi-chain strategies or join the validator network to earn protocol rewards.

<CardGroup cols={2}>
  <Card title="Write a Hook" icon="code" href="/build/hooks">

  </Card>
  <Card title="Become a Validator" icon="shield-check" href="/build/become-a-validator">

  </Card>
</CardGroup>
