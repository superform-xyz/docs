# Become a Validator

<Note>
The validator onboarding program is currently invite-only. This page will be updated with full documentation as the validator network decentralizes. If you are interested in becoming a validator, reach out to the Superform team via [Discord](https://discord.gg/superform).
</Note>

## What Validators Do

Validators secure SuperVault price-per-share (PPS) updates by independently computing PPS off-chain, signing EIP-712 attestations, and submitting them to the `ECDSAPPSOracle` contract. The oracle accepts an update only after at least two-thirds of weighted validator stake reaches consensus.

## Requirements

- **$UP stake** - Validators must bond $UP tokens to join the validator set
- **Infrastructure** - Run a monitoring node that tracks vault positions and computes PPS
- **Uptime** - Consistent participation is required; insufficient participation triggers slashing

## Economics

- Validators earn a share of $UP upkeep payments from vault curators
- Misreporting is deterred through slashing of bonded $UP
- An insurance fund backstops depositors against oracle failures
